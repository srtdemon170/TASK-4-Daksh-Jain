# General Knowledge Quiz

## Project Overview

General Knowledge Quiz is a Python console application that asks the user three general knowledge questions, checks each answer, calculates the score, and displays the final result.

The project demonstrates the use of variables, user input, conditional statements, string methods, and score tracking.

---

## Features

- Asks three General Knowledge questions.
- Accepts user input.
- Ignores uppercase/lowercase differences.
- Removes unnecessary spaces using `.strip()`.
- Calculates the total score.
- Displays personalized feedback based on the score.

---

## How to Run

1. Install Python 3.
2. Save the file as `general_knowledge_quiz.py`.
3. Open Terminal or Command Prompt.
4. Run:

```bash
python general_knowledge_quiz.py
```

## Project Workflow

1. Display quiz title.
2. Initialize score to 0.
3. Ask Question 1 and check the answer.
4. Ask Question 2 and check the answer.
5. Ask Question 3 and check the answer.
6. Increase the score for each correct answer.
7. Display the final score.
8. Show a performance message.
